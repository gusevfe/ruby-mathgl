Ruby/MathGL

Ruby wrapper of MathGL - library for scientific data visualization made using SWIG.

(This package is under construction.)

Usage

Write to PNG

require 'mathgl'

prc = proc{
  x = (0..100).map{|i| i*0.1}
  y = x.map{|i| Math.sin(i)}
  set_ranges(0,10,-1,1)
  box
  axis
  plot(x,y)
}

gr = MathGL::MglGraph.new
gr.instance_eval(&prc)
gr.write_png("test.png","",false)
